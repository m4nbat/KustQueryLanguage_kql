# T1036.003: Masquerading - Rename System Utilities Detection

## Query Information

#### MITRE ATT&CK Technique(s)

| Technique ID | Title    | Link    |
| ---  | --- | --- |
| T1036.003 | Masquerading: Rename System Utilities | [Rename System Utilities](https://attack.mitre.org/techniques/T1036/003/) |

#### Description
Detects processes executing with command lines or behaviors associated with different system utilities, indicating potential masquerading. Looks for suspicious PowerShell-like commands in unexpected processes.

#### Risk
Renaming system utilities allows adversaries to evade process name-based detection. This technique was observed in various threat campaigns where malware renames legitimate Windows binaries.
